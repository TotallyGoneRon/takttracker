Create `src/lib/health-index.ts` with the following:

**Interface:**
```typescript
export interface HealthIndex {
  score: number;       // 0-100 composite
  ppc: number;         // 0-100 percent
  spi: number;         // ratio around 1.0
  compression: number; // ratio around 1.0
  label: string;       // 'On Track' | 'At Risk' | 'Behind Schedule'
}
```

**Main function:** `export async function calculateHealthIndex(planId: number): Promise<HealthIndex>`

**PPC calculation (per D-01, matches scorecard-service.ts pattern):**
- Query all tasks for the plan: `db.select().from(tasks).where(eq(tasks.takt_plan_id, planId))`
- `today = getToday()` (Mountain Time YYYY-MM-DD)
- `shouldBeDone = tasks where planned_end <= today`
- `completedOfShouldBeDone = shouldBeDone.filter(t => t.status === 'completed')`
- `ppc = shouldBeDone.length === 0 ? 100 : Math.round((completedOfShouldBeDone.length / shouldBeDone.length) * 100)`

**SPI calculation (per Claude's discretion - standard EVM):**
- `totalTasks = allTasks.length`
- `completedTasks = allTasks.filter(t => t.status === 'completed').length`
- `ev = completedTasks / totalTasks` (earned value as fraction)
- `pv = shouldBeDone.length / totalTasks` (planned value as fraction)
- `spi = pv === 0 ? 1.0 : ev / pv` (default 1.0 when nothing planned yet)
- Cap SPI display at 1.5 for normalization but store raw value

**Compression ratio calculation (per research - simplified proxy):**
- `incompleteTasks = allTasks.filter(t => t.status !== 'completed')`
- `pastStartTasks = incompleteTasks.filter(t => t.planned_start && t.planned_start <= today)`
- `compression = incompleteTasks.length === 0 ? 1.0 : 1 - (pastStartTasks.length / incompleteTasks.length)`
- This captures "what fraction of remaining work is already behind its planned start"
- Value of 1.0 means no compression (no tasks past start), 0.0 means fully compressed

**Health Index composite (per D-01 - three-factor formula):**
- Weights: `w1 = 0.4` (PPC), `w2 = 0.35` (SPI), `w3 = 0.25` (compression)
- Normalize PPC: already 0-100
- Normalize SPI: `Math.min(spi, 1.2) / 1.2 * 100` (cap at 1.2, scale to 0-100)
- Normalize compression: `Math.min(compression, 1.5) / 1.5 * 100` (cap at 1.5, scale to 0-100)
- `score = Math.round(w1 * ppcNorm + w2 * spiNorm + w3 * compressionNorm)`

**Label logic:**
- `score >= 80` -> `'On Track'`
- `score >= 60` -> `'At Risk'`
- `score < 60` -> `'Behind Schedule'`

**Edge case (no tasks due yet):**
- When `shouldBeDone.length === 0` AND `completedTasks === 0`: return `{ score: -1, ppc: 100, spi: 1.0, compression: 1.0, label: 'No tasks due yet' }`
- Score of -1 signals "N/A" to the dashboard

**Also export a color helper:**
```typescript
export function getHealthColor(score: number): string {
  if (score < 0) return 'text-gray-500 bg-gray-50';
  if (score >= 80) return 'text-green-600 bg-green-50';
  if (score >= 60) return 'text-yellow-600 bg-yellow-50';
  return 'text-red-600 bg-red-50';
}
```
